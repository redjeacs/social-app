# 🌐 Full-Stack Real-Time Social Application

A modern, real-time social media platform featuring secure authentication, live instant messaging, post feeds, media storage, and relational user networking. 

ℹ️ **Live App Link:** [Deploy on Vercel](https://vercel.app)
ℹ️ **Backend API Link:** [Deploy on Render](https://onrender.com)

---

## ✨ Features

- **🔐 Secure Authentication:** Third-party OAuth and custom login flows powered by Passport.js.
- **💬 Real-Time Messaging:** Instant room-based messaging with file-sharing integrations powered by Socket.io.
- **📸 Media Cloud Storage:** Base64 dynamic media streaming directly uploaded and hosted on Cloudinary.
- **📝 Feed & Interactions:** Create posts, track user relationships, and navigate dynamic user profiles.

---

## 🛠️ Tech Stack

### Frontend
- **Framework:** React.js (Vite)
- **Deployment:** Vercel
- **State & Network:** Socket.io-Client, Fetch API / Axios

### Backend
- **Runtime:** Node.js & Express
- **Real-Time:** Socket.io (HTTP Wrap Wrapper Server architecture)
- **Database:** PostgreSQL (with custom relational abstractions)
- **Storage & Auth:** Cloudinary SDK, Passport.js
- **Deployment:** Render (Web Service architecture)
